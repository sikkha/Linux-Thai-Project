FAQ

  • ทำไมโฮมเพจนี้ถึงมี Underconstruction เยอะจัง

    • ตอบ : เราต้องขออภัยท่านที่ต้องการหาข้อมูลที่สนใจ แล้วเจอ Underconstruction แทน เรากำลัง
    รีบเร่งจัดทำข้อมูลหลายอย่างที่เป็นประโยชน์อยู่ครับ ขอเวลาอีกสักนิดแล้วเราจะทำโฮมเพจนี้ให้มีข้อมูล
    ของลีนุกซ์ที่น่าสนใจครับ


  • จะแก้ไขขั้นตอนการบูตลีนุกซ์ที่ใช้ lilo ต้องทำอย่างไร

    • ตอบ : ให้คุณเข้าไปแก้ไขที่ไฟล์ lilo.conf ซึ่งจะอยู่ที่ไดเรกทอรี /etc ก่อนการ แก้ไขใดๆขอแนะนำ
    ให้ทำการแบคอัพไฟล์เหล่านี้ไว้ก่อน เมื่อแก้ไขเสร็จแล้วให้ใช้คำสั่ง /sbin/lilo เพื่อทำการเขียนบูตพาร์ติชันใหม่

    ตัวอย่างของไฟล์ /sbin/lilo และคำอธิบาย
    #lilo ทำการบูตจาก MBR (Master Boot Record) ของฮาร์ดดิสก์ตัวแรก
    boot=/dev/hda

    #แผนที่ที่ "บูตโหลดเดอร์" ใช้บอกตำแหน่งเคอร์เนล
    map=/boot/map

    #lilo ทำการเขียนไฟล์นี้ลงไปใน MBR
    install=/boot/boot.b

    #จำกัดเวลาที่ให้รอรับการเลือกบูต 50 วินาที ถ้าเกินนี้ ระบบจะบูตเข้าสู่โอเอสชุดแรก
    #โดยอัตโนมัติ (ในที่นี้คือ linux)
    prompt
    timeout=50

    #คำอธิบายการบูตส่วนลีนุกซ์
    image=/boot/vmlinuz

    #ชื่อที่จะต้องใส่ที่พร้อมพท์ของ lilo (สามารถกดปุ่ม tab เพื่อดูรายชื่อได้)
    label=linux

    #บอกพาร์ติชันที่จะให้ลีนุกซ์ทำการเมาท์ขึ้นมาเป็น root พาร์ติชัน
    root=/dev/hda2
    initrd=/boot/initrd
    read-only

    #ส่วนของดอสพาร์ติชัน
    other=/dev/hda1
    label=dos
    table=/dev/hda



  • จะทำอย่างไรเพื่อเพิ่มดิสก์ให้กับระบบลีนุกซ์

    • ตอบ : เนื่องจากระบบลีนุกซ์มีการอ้างโครงสร้างของระบบไฟล์เป็นแบบไดเรกทอรีเดี่ยว
    จะไม่มีลักษณะเป็นหลายๆไดรฟ์เหมือนในดอส การเพิ่มดิสก์ในลีนุกซ์จึงต้องใช้วิธีกำหนด
    ควบคู่กับไดเรกทอรีในระบบ ดังรูป
    /---+ (/dev/hda1)
    |
    +--- usr
    +--- home --+
		+-- usr1
		+-- usr2 (/dev/hdb2)
    จากรูปจะเห็นว่าเราไม่สามารถนำเอาเนื้อที่ของดิสก์มาเพิ่มเติมให้กับไดเรกทอรีที่มี
    ไฟล์ในไดเรกทอรีนั้นอยู่แล้วได้ จะต้องเป็นไดเรกทอรีว่างๆเท่านั้น
    สรุปวิธีการเพิ่มดิสก์
    1. ทำการแบ่งพาร์ติชั่นและเปลี่ยนพาร์ติชั่น ID ให้เป็นของ ID ลีนุกซ์ โดยใช้ fsdisk
    2. ฟอร์แมตพาร์ติชั่นที่แบ่งใหม่โดยใช้คำสั่ง mkfs
      /sbin/mkfs -t ext2 /dev/hdb2 [partition's size]
    3. สร้างไดเรกทอรีเตรียมไว้สำหรับการเมาท์ดิสก์เข้ามา ดังในรูปจะทำการเมาท์
      มาไว้ใต้ /home/usr2
    4. เมาท์ดิสก์เข้ามาไว้ใต้ไดเรกทอรี
      /sbin/mount -t ext2 /dev/hdb2 /home/usr2
    หลังจากทำการเมาท์เสร็จแล้ว คุณจะสามารถใช้พื้นที่ของพาร์ติชั่น /dev/hdb2 ได้ภาย
    ใต้ไดเรกทอรีของ /home/usr2
    หากต้องการจะให้ทำการเมาท์พาร์ติชั่นโดยอัตโนมัติตั้งแต่ตอนเริ่มบูตระบบ ให้คุณเข้า
    ไปแก้ไขในไฟล์ /etc/fstab ตามรูปแบบนี้
    [device] [mountpoint] [filesystemtype] [option] [dump] [fsckorder]
    เช่น
    /dev/hdb2 /home/usr2 ext2 defaults 1 1



คุณสามารถส่งคำถามที่สงสัย ไม่ว่าจะเป็นเรื่องราวเกี่ยวกับลีนุกซ์ หรือเกี่ยวกับโครงการฯ หรือเกี่ยวกับโฮมเพจ
