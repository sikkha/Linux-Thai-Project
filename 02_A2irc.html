คุยกันแบบออนไลน์โดยใช้ irc

อินเทอร์เน็ตนั้นนอกจากจะให้ข้อมูลข่าวสารผ่านทาง เวิร์ด ไวด์ เว็บ (WWW) และยูสเน็ตนิวส์ (Usenet News) แล้ว คุณยังอาจจะสามารถแลกเปลี่ยนความรู้ข่าวสารกับนักท่องอินเทอร์เน็ตได้จากทาง irc (ไอ อาร์ ซี) หรือ Internet Relay Chat อีกช่องทางหนึ่ง

ข้อดีของ irc นั้นคือเราสามารถพูดคุยโต้ตอบกันได้ในทันที เปรียบเสมือนการได้พูดคุย กับคนจริงๆเพียงแต่เราไม่เห็นหน้าตาของคู่สนทนาเท่านั้น นอกจากนี้เรายังอาจสามารถที่จะพูดคุยได้กับคนมากกว่าหนึ่งคน ทำให้สามารถสนทนากันเป็นกลุ่มใหญ่ๆได้

ลักษณะการใช้ irc นั้นเราจะต้องทำการเชื่อมต่อไปยังเซอร์ฟเวอร์ หรือคอมพิวเตอร์ที่ ให้บริการ irc สำหรับเซอร์ฟเวอร์ที่มีผู้นิยมต่อเข้าไปจำนวนมากนั้นเห็นจะเป็นเซอร์ฟเวอร์ของบริษัทไมโครซอฟท์ คือ irc1.msn.com หรือ comicsrv1.microsoft.com เมื่อเชื่อมต่อเข้าไปได้แล้ว จะพบว่ามีห้อง (room หรือ channal) ปรากฏอยู่มากมาย คุณจะต้องเลือกเข้าไปในห้องสักห้องหนึ่ง จากนั้นคุณก็จะพบสังคมของมนุษย์ในอีกแบบหนึ่ง ที่ซึ่งเราสามารถจะพูดคุยสนทนากันได้แม้ว่าจะอยู่กันคนละซีกโลกก็ตาม

มารยาทในการใช้งาน irc

ถึงแม้ว่าคุณจะเห็นว่าการใช้งาน irc จะไม่ได้เห็นหน้าตาของคู่สนทนาแต่ละคน รวมทั้ง ยังสามารถปกปิดข้อมูลส่วนตัวได้เป็นอย่างดีด้วย แต่พึงตระหนักไว้เสมอว่าเบื้องหลัง ชื่อนิกเนม หรือนามแฝงที่ประกาศไว้กับเซอร์ฟเวอร์เพื่อแทนตัวผู้ใช้แต่ละคนนั้น จริงๆแล้ว ก็เป็นคนธรรมดาที่มีชีวิตจิตใจที่เราพบปะกันได้ทั่วไปในสังคมนั่นเอง ดังนั้นมารยาทที่ดีจึงเป็นสิ่งที่คาดหวังของคู่ร่วมสนทนา อย่างไรก็ดีคุณไม่ควรมอบข้อมูลส่วนตัวของคุณแก่คนที่คุณยังไม่รู้จักในตอนเริ่มแรก จนกว่าจะได้รู้จักมักคุ้นกันอย่างดีเสียก่อน

Client

irc client หรือโปรแกรม irc ที่อยู่ทางฝั่งของผู้ใช้ มีให้เลือกใช้งานมากมายหลายตัว แต่สำหรับแพลตฟอร์ม ดอส/วินโดวส์ แล้ว โปรแกรมยอดนิยมก็คือ pirch และ mirc ส่วนโปรแกรมที่มาทีหลังมาแรงอย่าง Microsoft commic chat อาศัยอินเทอร์เฟสที่เข้าใจง่ายและการอินเทอร์เฟสที่เป็นแบบ "อินเทอร์เฟสทางสังคม" (social interface) โดยอนุญาตให้เราสามารถเลือกเป็นตัวการ์ตูนและออกท่าทางในการสนทนาได้

สำหรับบนลีนุกซ์แล้วมีโปรแกรม irc client ให้เลือกมากมายเช่นกัน นับตั้งแต่ ircII, zircon, kirc เป็นต้น โปรแกรมเหล่านั้นมีทั้งที่ใช้งานเป็นแบบคอมมานด์ไลน์และแบบ GUI คุณสามารถเลือกใช้งานได้ตามถนัดของคุณ

คำสั่งพื้นฐาน และคำย่อที่นิยมใช้

ถึงแม้ว่าจะมี irc client ที่ใช้งานเป็นแบบ GUI แต่ irc client แทบทุกตัวก็มีความสามารถในการใช้งานในโหมดคอมมานด์ไลน์ได้ โดยคำสั่งพื้นฐานส่วนใหญ่จะใกล้เคียงกัน คำสั่งพื้นฐานที่จำเป็นในการใช้งาน irc client มีดังต่อไปนี้
/LIST สั่งแสดงชื่อห้องที่มีใน server
/NICK [new nick name] เปลี่ยนนามแฝงใหม่
/NAME [room's name] สั่งแสดงรายชื่อของผู้ที่อยู่ในห้อง
/WHOIS [nick name] แสดงข้อมูลของผู้ใช้ที่ระบุ (ใส่นามแฝงของผู้ต้องการจะติดต่อ)
/PING [nick name] ทดสอบความเร็วการตอบรับของ โปรแกรม client ของผู้ใช้ที่ระบุ
/JOIN [room's name] เข้าไปสนทนาในห้องที่ระบุ (ใส่ชื่อห้อง)
/LEAVE [room's name] ออกจากห้องที่กำลังสนทนาอยู่
/MSG [nick name] [message] ส่งข้อความที่เจาะจงถึงผู้รับ ข้อความที่ส่งไปจะไม่ปรากฏให้ผู้ใช้ทั่วไปได้เห็น จะเห็นได้เฉพาะผู้ที่เราระบุชื่อเท่านั้น ใน commic chat จะเรียกว่า whisper (กระซิบ)
/SERVER [server name] แสดงและเปลี่ยนเครื่องเซอร์ฟเวอร์ที่ใช้งาน
/QUIT ออกจากโปรแกรม
ในกรณีที่คุณเป็นผู้เข้าไปในห้องเป็นคนแรกหรือเป็นคนสร้างห้องสนทนาขึ้นมา คุณจะได้รับสถานะเป็น "ผู้ดูแล" (operator) สถานภาพของผู้ดูแลจะสามารถแต่งตั้งคู่ร่วมสนทนาในห้องคนอื่นให้เป็น "ผู้ดูแล" ขึ้นมาอีกก็ได้ หรือแม้แต่จะไล่คู่สนทนาที่มีมารยาทไม่เหมาะสมให้ออกจากห้องไปได้ คำสั่งสำหรับ "ผู้ดูแล" มีดังต่อไปนี้
/OPER [nick name] แต่งตั้งคู่สนทนาที่ระบุเป็น "ผู้ดูแล"
/DEOP [nick name] ยกเลิกคู่สนทนาที่เป็น "ผู้ดูแล" กลับสู่สถานะปกติ
/KICK [nick name] ไล่คู่สนทนาที่ระบุออกจากห้องสนทนา
ระวังอย่าแต่งตั้ง "ผู้ดูแล" ให้กับคนที่คุณยังไม่ไว้ใจ เพราะเขาอาจเข้าครอบครองห้องสนทนาของคุณได้

ในการสนทนาผ่าน irc เพื่อไม่ให้ต้องเสียเวลากับการพิมพ์คำพูดยาวๆ ชาว irc จึงมีสัญลักษณ์ย่อเพื่อสื่อความหมายแทนคำพูดยาวๆ รวมทั้งแทนอารมณ์ของผู้พูดได้ด้วยสัญลักษณ์เหล่านี้เรียกว่า smiley สัญลักษณ์เหล่านี้ต้องตะแคงข้างดู
:-) หยอกเล่น, ยิ้มสบายใจ
:-( เศร้า
:~-( เศร้ามาก (มีน้ำตาไหล)
:-| หน้าไร้ความรู้สึก
8-) ผู้พูดใส่แว่น
:^) ผู้พูดจมูกโต
(-: ถนัดซ้าย
(g) Grin (small) ฉีกยิ้มเล็กๆ
(G) Grin (Big) ฉีกยิ้มกว้าง
LOL Laughing out loud ขำมาก
ROFL Rolling on the floor laughing ขำมากๆ
IMHO In my humble (or honest) opinion ในความเห็นอันต่ำต้อย(หรือซื่อๆ)ของข้าพเจ้า
BTW By The way ดังนั้น
TIA Thanks in advance ขอขอบคุณล่วงหน้า
ถ้าคุณได้ลองเข้าไปพูดคุยใน irc กับชาวต่างชาติคุณจะพบว่าทักษะการสื่อสารภาษาอังกฤษของคุณเพิ่มขึ้นอย่างรวดเร็วทีเดียวแม้ว่าแรกๆอาจต้องเปิดดิกชันนารีกันจนเมื่อยมือบ้าง ผู้เขียนขอสนับสนุนให้คุณลองสนทนากับชาวต่างชาติ อย่างน้อยก็จะได้เป็นการฝึกภาษาและเป็นการเรียนรู้ชีวิตความเป็นอยู่และวัฒนธรรมของกันและกัน ลองเลือกห้องที่มีชื่อและหัวข้อที่ตรงกับความสนใจของคุณ แล้วก็อย่ารอช้ารีบเข้าไป /join กันเลย :-)

ขอเชิญร่วมห้องสนทนาของ #ThaiLinux

ขอเรียนเชิญท่านที่มีความสนใจในระบบปฏิบัติการลีนุกซ์ และผู้ที่สนใจจะร่วมเป็นอาสาสมัครในโครงการลีนุกซ์ภาษาไทย ทั้งการเขียนเอกสารภาษาไทย และการพัฒนาซอฟท์แวร์ ร่วมพบปะแลกเปลี่ยนความคิดเห็นผ่านทางอินเทอร์เน็ต 
server: comicsrv1.microsoft.com, irc1.msn.com, irc2.msn.com, irc3.msn.com
channel: #ThaiLinux
Online Time: Saturday-Sunday 22:00-01:00 Bangkok Time (GMT+7), 15:00-18:00 (GMT)

Client ircII ฉบับภาษาไทย

สำหรับท่านที่ใช้งานลีนุกซ์และรันระบบ X อยู่ด้วย ขอให้ดาวน์โหลด ircII ฉบับแก้ไข source code ให้สามารถใช้งานภาษาไทยได้ที่นี่ สำหรับ ircII เวอร์ชั่นนี้จะล็อกอินเข้า irc server ของไมโครซอฟท์เป็นดีฟอลท์เซอร์ฟเวอร์ รวมทั้งไม่แสดงข้อมูลที่เกี่ยวกับชื่อล็อกอินและข้อมูลของผู้ใช้ การติดตั้งและใช้งาน ircII ฉบับภาษาไทย

ขอให้คุณคลี่ไฟล์ irc.gz ลงในไดเรกทอรีปัจจุบันของคุณ (หรือทับไฟล์ irc เก่าใน /usr/bin/irc) แต่แนะนำให้คุณแบ็คอัพโปรแกรม irc เวอร์ชั่นเดิมของคุณไว้ด้วย ใช้คำสั่งดังนี้ 
$ gunzip irc.gz
เรียกโปรแกรมพร้อมทั้งกำหนดนามแฝงของคุณ โดยใช้คำสั่งดังต่อไปนี้
กรณีคลี่ไฟล์ทับ /usr/bin/irc 
$ irc mynick
กรณีคลี่ไฟล์ลงในไดเรกทอรีปัจจุบัน 
$ ./irc mynick
หมายเหตุ:
  • คุณจะต้องใช้งานโปรแกรมใน Thai Xterm บนระบบ X window
  • โปรแกรม ircII ฉบับภาษาไทยนี้ยังมีข้อผิดพลาดอยู่บ้าง กล่าวคือการลบตัวอักษรจะทำให้เกิดขยะขึ้น และดูเสมือนว่าไม่มีการลบตัวอักษรเกิดขึ้น
